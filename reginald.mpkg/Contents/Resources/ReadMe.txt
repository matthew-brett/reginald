Say hello to Reginald, a heap of scientific python modules.

Reginald contains the scipy stack : see http://scipy.github.com/stackspec.html

You will need to install the Python.org python via the mpkg installer before installing Reginald.

This version of Reginald needs the installer at:

http://python.org/ftp/python/2.7.3/python-2.7.3-macosx10.6.dmg

The full list of packages is:

* numpy
* scipy
* matplotlib
* ipython
* ipython-notebook
* pandas
* sympy
* nose
* python-readline
* distribute
* pip
* tornado
* pyzmq
* nipy
* nibabel
* pydicom
